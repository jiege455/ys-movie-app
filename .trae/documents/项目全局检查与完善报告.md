# 完善项目并进行全局检查

## 1. 首页 (HomePage) 动态 Tab 支持
已重构 `home_page.dart`，使其能够根据后台配置 (`app_tab_setting_list`) 动态加载特殊的分类页面：
- **排行榜 (RankingPage)**: 当后台配置 `rank` 时显示。
- **排期表 (WeekPage)**: 当后台配置 `week` 时显示。
- **专题 (TopicPage)**: 当后台配置 `topic` 时显示。
- **普通分类**: 继续支持电影、电视剧等标准分类。

## 2. 功能页面完善
已确认并完善以下页面，确保与插件后台功能对应：
- **排行榜 (`ranking_page.dart`)**: 支持日榜、周榜、月榜切换。
- **排期表 (`week_page.dart`)**: 显示周一至周日的更新列表。
- **专题页 (`topic_page.dart`)**: 展示专题列表及详情。
- **用户中心 (`user_center_pages.dart`)**: 包含会员购买 (`VipPage`)、积分记录 (`PointsPage`)、推广邀请 (`InvitePage`)。
- **反馈中心 (`feedback_center_page.dart`)**: 包含留言反馈、求片、消息中心。

## 3. 视频详情页 (`detail_page.dart`) 优化
- **竖屏视频支持**: 自动检测视频比例，如果是竖屏视频（如短剧），播放器会自动调整为竖屏模式，全屏时锁定为竖屏，避免画面被压缩。
- **评论功能**: 实现了评论列表加载和发表评论功能（需登录）。
- **交互优化**: 简介支持点击展开，选集支持横向滑动和弹窗查看全部。
- **播放源**: 修复了播放源名称显示问题，优先读取插件配置的名称。

## 4. 个人中心 (`profile_page.dart`) 链接
- 已更新个人中心 UI，将 "会员中心"、"我的积分"、"推广记录"、"求片反馈" 等按钮正确链接到对应的功能页面。

## 5. API 服务 (`api.dart`)
- 补全了所有必要的接口方法，包括 `getAppInit` (解析 Tab 配置), `getComments`, `sendComment`, `getUserVipCenter`, `buyVip` 等，确保前端能与后端插件完全交互。
